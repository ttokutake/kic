# KIC (Keep It Clean)

## Feature

- "kic" prevents someone from making a mess of directories.
- For registered directories, "kic" moves "garbage" files into "trash box" by using `cron`.
- "garbage" is file which is not specified as "requirement" in "kic".

## Commands

- `help`
- `init`
- `set`
- `sweep`
- `burn`
- `start`
- `end`
- `destroy`
