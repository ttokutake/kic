# KIC (Keep It Clean)

## Feature

- "kic" prevents someone from making a mess of directories.
- For registered directories, "kic" moves "dust" files into "dust box" by using `cron`.
- "dust" is file which is not specified as "requirement" in "kic".

## Commands

- `init`
- `set`
- `sweep`
- `burn`
- `start`
- `end`
- `destroy`

## Option

- `-h`, `--help`
